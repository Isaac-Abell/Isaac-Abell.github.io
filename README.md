## Getting Started

### 1. Install Dependencies

Before running the project, install the required packages:

```bash
npm install
```

### 2. Run Locally

To start the development server:

```bash
npm run dev
```

### 3. Deploy to Website

To build and deploy the site:

```bash
npm run deploy
```
